Pin configuration
-----------------------------------------------------------
The Soil Moisture sensor (MHI sensor series) uses the following connection to Arduino

| Soil Moisture  | Arduino Pin |
| ------------- | ------------- |
| VCC+ | 5V VCC+ |
| GND | Gnd |
| A0 | A0 |
